# TASK-04: Add Command

## Dependencies: TASK-03

## Description

Implement the `add` command that creates a new todo item and saves it to storage.

## Acceptance Criteria

- [ ] `todo add <title>` creates a new todo with the given title
- [ ] New todo has `completed: false`, auto-generated `id`, and current timestamp
- [ ] Prints confirmation message with the todo id after successful creation
- [ ] Validates that title is provided and non-empty
